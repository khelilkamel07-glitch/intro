Chapitre 01

C'est quoi l'Informatique ?

L'informatique est la science du traitement automatique de l'information. Elle regroupe tout ce qui concerne les ordinateurs : le matériel, les logiciels, les réseaux et les algorithmes.

🧠
Algorithme
Logic & Instructions

Suite d'instructions précises et ordonnées pour résoudre un problème. C'est la base de tout programme informatique.

Séquence d'étapes logiques
Entrée → Traitement → Sortie (modèle IPO)
Peut être écrit en pseudocode ou en organigramme
Fondamental
🌐
Réseau
Network & Communication

Ensemble d'ordinateurs et d'appareils connectés pour partager des données et des ressources.

LAN (réseau local), WAN (réseau étendu)
Protocoles : TCP/IP, HTTP, FTP
Internet = réseau des réseaux mondiaux
Infrastructure
💻
Système informatique
Hardware + Software

Ensemble formé par le matériel (hardware) et les logiciels (software) qui travaillent ensemble pour traiter l'information.

Hardware : partie physique visible et tangible
Software : programmes, OS, applications
Firmware : logiciel intégré dans le matériel
Système
Chapitre 02

Composants d'un Ordinateur

L'ordinateur est composé de plusieurs éléments matériels qui travaillent ensemble. Voici les composants essentiels avec des exemples réels.

🖥️
CPU – Processeur
Central Processing Unit

Le cerveau de l'ordinateur. Il exécute toutes les instructions des programmes et effectue les calculs.

Exemples : Intel Core i9, AMD Ryzen 9
Mesure : fréquence en GHz (ex. 3.8 GHz)
Plus le GHz est élevé = plus c'est rapide
Composé de millions de transistors
Matériel principal
🧩
RAM – Mémoire vive
Random Access Memory

Mémoire temporaire ultra-rapide. Elle stocke les données des programmes en cours d'utilisation.

Exemples : DDR4 8Go, DDR5 16Go
Se vide quand on éteint l'ordinateur
Plus de RAM = plus d'applications simultanées
Différente du stockage (SSD/HDD)
Mémoire
🔌
Carte Mère
Motherboard / Mainboard

Le circuit imprimé principal. Elle connecte tous les composants entre eux et permet leur communication.

Exemples : ASUS ROG, MSI MAG, Gigabyte
Contient le BIOS (démarrage du PC)
Ports : USB, PCIe, SATA, RAM slots
Format : ATX, Micro-ATX, Mini-ITX
Infrastructure
💽
SSD / HDD
Stockage permanent

Stockage permanent des données : système d'exploitation, fichiers, programmes. Les données restent même après extinction.

SSD (Solid State Drive) : rapide, silencieux, cher
HDD (Hard Disk Drive) : moins cher, plus lent
Capacité : 256 Go → plusieurs To
NVMe M.2 : SSD ultra-rapide nouvelle génération
Stockage
🎮
GPU – Carte Graphique
Graphics Processing Unit

Spécialisée dans le traitement des images et des calculs parallèles massifs. Essentielle pour les jeux vidéo et l'IA.

Exemples : NVIDIA RTX 4090, AMD RX 7900
Milliers de cores pour le calcul parallèle
VRAM : mémoire dédiée à la carte graphique
Utilisée aussi pour le Machine Learning / IA
Graphique
⚡
Alimentation (PSU)
Power Supply Unit

Fournit l'énergie électrique à tous les composants de l'ordinateur. Convertit le courant alternatif (AC) en courant continu (DC).

Puissance en Watts : 500W, 750W, 1000W
Certification : 80 PLUS Bronze/Gold/Platinum
Modulaire ou non-modulaire
Protections : surtension, surintensité, court-circuit
Énergie
⌨️
Périphériques I/O
Input / Output Devices

Permettent à l'utilisateur d'interagir avec l'ordinateur. Divisés en entrée (input) et sortie (output).

Entrée (Input) : clavier, souris, microphone, scanner
Sortie (Output) : écran, imprimante, haut-parleurs
Entrée+Sortie : écran tactile, casque avec micro
I/O
🖥️
Système d'exploitation
Operating System – OS

Logiciel principal qui gère le matériel et les applications. Il fait l'interface entre l'utilisateur et la machine.

Exemples : Windows 11, macOS, Linux Ubuntu
Gère la mémoire, les processus, les fichiers
Interface graphique (GUI) ou ligne de commande (CLI)
Android et iOS = OS pour mobiles
Logiciel système

💡 Moyen mnémotechnique : Pense à l'ordinateur comme un restaurant. Le CPU = le chef cuisinier (qui fait tout le travail), la RAM = le plan de travail (espace temporaire), le SSD = le garde-manger (stockage permanent), la carte mère = la cuisine (relie tout le monde), le PSU = l'électricité, et toi (l'utilisateur) = le client avec clavier/souris.

Chapitre 03

Développement Web

Le développement web consiste à créer des sites et applications accessibles via un navigateur. Il se divise en Front-end (ce que voit l'utilisateur) et Back-end (serveur & base de données).

🏗️ HTML – Structure

HyperText Markup Language. Définit la structure et le contenu d'une page web avec des balises.

<!DOCTYPE html> <html> <head> <title>Ma Page</title> </head> <body> <h1>Bonjour !</h1> <p>Club CNFP</p> </body> </html>
Importance
🎨 CSS – Style

Cascading Style Sheets. Gère la mise en forme : couleurs, polices, animations, mise en page responsive.

body { background: #1a2040; color: white; font-family: 'Arial'; } h1 { color: gold; font-size: 2rem; }
Importance
⚙️ JavaScript – Interactivité

Langage de programmation du web. Rend les pages interactives : boutons, animations, appels API, logique.

// Afficher un message function saluer() { let nom = "CNFP"; alert("Bonjour " + nom); } document.getElementById("btn") .onclick = saluer;
Importance
🖥️ Back-end – Serveur

Côté serveur invisible pour l'utilisateur. Traite les données, gère l'authentification, communique avec la base de données.

Langages : PHP, Python, Node.js, Java
Frameworks : Laravel, Django, Express.js
API REST : échange de données JSON
Serveurs web : Apache, Nginx
Importance
🗄️ Base de données

Stocke et organise les données de façon structurée. Le back-end interroge la BDD pour récupérer ou enregistrer des informations.

-- Requête SQL simple SELECT nom, prenom FROM etudiants WHERE universite = 'Oran' ORDER BY nom ASC;
Importance
📱 Responsive Design

Adaptation automatique du site à tous les types d'écrans : mobile, tablette, desktop. Indispensable aujourd'hui.

Media queries CSS pour chaque taille d'écran
Mobile First : concevoir d'abord pour mobile
Flexbox et CSS Grid pour les mises en page
Frameworks : Bootstrap, Tailwind CSS
Importance
Chapitre 04

Comment fonctionne un site web ?

Quand tu tapes une URL dans ton navigateur, toute une chaîne d'événements se déclenche en moins d'une seconde.

Étape 01

L'utilisateur tape une URL

Ex : https://www.google.com — le navigateur (Chrome, Firefox…) va devoir trouver le serveur correspondant à ce nom.

Étape 02

Résolution DNS

Le DNS (Domain Name System) traduit le nom de domaine en adresse IP numérique. Comme un annuaire téléphonique : "google.com" → "142.250.74.46".

Étape 03

Connexion TCP/IP & Requête HTTP

Le navigateur se connecte au serveur et envoie une requête GET /index.html HTTP/1.1. HTTPS chiffre la communication avec SSL/TLS pour la sécurité.

Étape 04

Traitement côté serveur

Le serveur web (Apache/Nginx) reçoit la requête, exécute le code back-end (PHP, Python…), interroge la base de données si nécessaire, puis prépare la réponse.

Étape 05

Réponse : HTML + CSS + JS

Le serveur renvoie les fichiers avec un code de réponse HTTP : 200 OK (succès), 404 (page non trouvée), 500 (erreur serveur).

Étape 06

Rendu par le navigateur

Le navigateur parse le HTML, applique le CSS, exécute JavaScript. Il construit le DOM (Document Object Model) et affiche la page finale à l'utilisateur.

🔐 HTTP vs HTTPS : HTTP envoie les données en clair (dangereux). HTTPS les chiffre avec SSL/TLS. Le cadenas 🔒 dans la barre d'adresse = connexion sécurisée. Toujours utiliser HTTPS pour les sites avec des mots de passe ou données personnelles.
